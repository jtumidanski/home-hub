# Layout Components

This directory contains layout components used throughout the admin portal.

## Components

- **Header.tsx** - Top navigation bar with user menu and branding
- **Sidebar.tsx** - Side navigation with menu items
- **Footer.tsx** - Footer with copyright and links

## Usage

```tsx
import { Header } from "@/components/layout/Header";
import { Sidebar } from "@/components/layout/Sidebar";
import { Footer } from "@/components/layout/Footer";
```

These components are typically used in the root layout (`app/layout.tsx`).
