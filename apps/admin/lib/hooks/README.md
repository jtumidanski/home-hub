# Custom Hooks

This directory contains custom React hooks for the admin portal.

## Examples

```
hooks/
  useAuth.ts          - Authentication state and functions
  useHouseholds.ts    - Household data and operations
  usePolling.ts       - 30-second polling for dashboard updates
```

## Guidelines

- Follow React hooks naming convention (use* prefix)
- Keep hooks focused and composable
- Document hook parameters and return values
